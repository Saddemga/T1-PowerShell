# T1-PowerShell
# les commandes unix et les commandes PowerShell équivalent


| Lunix    | PowerShell                  |
| -------- | -------                     |
|  cp      | Copy-Item                   |
|  rm      | Remove-Item                 |
|  cd      | Set-Location                |
|  mkdir   | New-Item -ItemType Directory|
|  man     | Get-Help                    |
| history  | Get-History                 |
| alias    |  Get-Help alias : tous les comandes qui est possible de faire liée a alias |
|  cat     | Get-content                 |
